# Go Practice Exercises

1. Multiplication Table
   - Input a number and print its multiplication table (0-10).

2. Countdown
   - Input a number and count down to 0.

3. Sum from 1 to N
   - Input a number and calculate the sum from 1 to that number.

4. Number Guessing Game
   - Keep asking the user to guess a secret number until they get it right.

5. Simple Calculator
   - Perform basic operations: add, subtract, multiply, and divide.
